# 要件・要求
## 要求
これは、複数人マルチプレイのゲームである。1人がお題に沿った絵を書き、残りのプレイヤーと1つのLLMがその絵を見て元のお題を推測する。本ソフトウェアはそのゲームのバックエンド部部分を担い、以下の機能を持つ

- お題を生成する
- イラストからお題を推測する

### お題を生成
レスポンスは、1つのトピックと1つのお題からなる。トピックは、出題するお題の「ジャンル」である。
例えば、「バナナ」というお題に対して「食べ物」のように、お題のヒントとなるような単語としてトピックを生成する。本アプリはトピックを手動で指定し、その中からランダムで選ばれたトピックからLLMを使ってお題を生成する。

### お題の推測
プレイヤーが描いたイラストとプレイヤーに与えられたトピックを入力として、お題を推測させる。LLMの返答は単語として出力させる


## 要件
REST APIサーバーを用いる。リクエスト・レスポンス共にJSONを利用して通信を行う

## お題の生成
入力は何も受け取らない。用意されたお題からランダムに1つを選択し、そのトピックに関係する単語をLLMに生成させる。生成させたお題とトピックを返す

## お代の推測
入力として画像とトピックを受け取る。トピックに関するお題をイラストから推測し、推測したトピックを返す。
