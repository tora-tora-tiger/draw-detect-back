# Draw Detect backend

## 動作確認済み環境

- node: 24.12.0
- pnpm: 10.26.1

## VSCode で開く場合

workspace を利用して設定を共有しています。  
`detect-draw-back.code-workspace` ファイルを VSCode で開いてください。

## 開発環境の起動方法

```
git clone https://github.com/tora-tora-tiger/draw-detect-back.git
pnpm i
pnpm dev
```
